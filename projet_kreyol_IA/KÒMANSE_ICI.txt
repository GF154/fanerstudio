KISA PWOJÈ SA A FÈ?
What does this project do?
═══════════════════════════════════════════════════════════════════════════

Pwojè sa a pèmèt ou:
This project allows you to:

✅ Tradui liv PDF nan Kreyòl Ayisyen
   Translate PDF books to Haitian Creole

✅ Kreye liv odyo (audiobook) an Kreyòl
   Create Creole audiobooks

✅ Kreye podcast an Kreyòl
   Create Creole podcasts

✅ Pataje fichye yo sou entènèt (lyen piblik)
   Share files online (public links)

✅ Trete plizyè liv an menm tan
   Process multiple books at once

╔══════════════════════════════════════════════════════════════════════════╗
║                  🚀 KÒMANSE RAPID (3 ETAP!)                              ║
║                  QUICK START (3 STEPS!)                                  ║
╚══════════════════════════════════════════════════════════════════════════╝

ETAP 1: ENSTALE PAKÈ YO / INSTALL PACKAGES
═══════════════════════════════════════════════════════════════════════════

Windows:
    setup_cloud.bat

Linux/Mac:
    chmod +x setup_cloud.sh
    ./setup_cloud.sh

Oswa / Or:
    pip install -r requirements.txt


ETAP 2: KONFIGIRE / CONFIGURE
═══════════════════════════════════════════════════════════════════════════

1. Kopye fichye config / Copy config file:
   cp cloud_env_template.txt .env

2. Modifye .env / Edit .env:
   nano .env    (Linux/Mac)
   notepad .env (Windows)

3. Ajoute non bucket ou / Add your bucket name:
   GCS_BUCKET_NAME=your-bucket-name

4. Otantifye / Authenticate:
   gcloud auth application-default login


ETAP 3: KÒMANSE! / START!
═══════════════════════════════════════════════════════════════════════════

OPSYON A - Meni Enteraktif (REKÒMANDE!)
OPTION A - Interactive Menu (RECOMMENDED!)

Windows:
    START.bat

Linux/Mac:
    chmod +x START.sh
    ./START.sh

Oswa / Or:
    python start_here.py


OPSYON B - Entèfas Web (PI FASIL!)
OPTION B - Web Interface (EASIEST!)

    streamlit run web_app.py

Oswa / Or:
    run_web.bat    (Windows)
    ./run_web.sh   (Linux/Mac)


OPSYON C - Liy Kòmand (POU EKSPE!)
OPTION C - Command Line (FOR EXPERTS!)

    python process_book_enhanced.py

╔══════════════════════════════════════════════════════════════════════════╗
║              📚 TRETE YON LIV / PROCESS A BOOK                           ║
╚══════════════════════════════════════════════════════════════════════════╝

1. Telechaje PDF ou a nan Google Cloud Storage
   Upload your PDF to Google Cloud Storage

2. Lance yon nan opsyon yo anwo a
   Run one of the options above

3. Suiv enstriksyon yo
   Follow the instructions

4. Tann pandan pwosesis la (ka pran kèk minit)
   Wait during processing (may take a few minutes)

5. Jwenn lyen piblik yo!
   Get public links!

Rezilta / Results:
   📘 Tèks tradui / Translated text (.txt)
   🔊 Audiobook (.mp3)
   🎙️ Podcast (.mp3)
   📋 Metadata (.json)

PWOBLÈM 1: "Bucket not found"
───────────────────────────────────────────────────────────────────────────
   Solisyon / Solution:
   - Verifye GCS_BUCKET_NAME nan .env
   - Check GCS_BUCKET_NAME in .env

PWOBLÈM 2: "Authentication failed"
───────────────────────────────────────────────────────────────────────────
   Solisyon / Solution:
   gcloud auth application-default login

PWOBLÈM 3: "Email pa voye / Email not sending"
───────────────────────────────────────────────────────────────────────────
   Solisyon / Solution:
   - Itilize Gmail App Password (pa modpas regilye)
   - Use Gmail App Password (not regular password)
   - https://myaccount.google.com/apppasswords

PWOBLÈM 4: "Module not found"
───────────────────────────────────────────────────────────────────────────
   Solisyon / Solution:
   pip install -r requirements.txt
