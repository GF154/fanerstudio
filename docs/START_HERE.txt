═══════════════════════════════════════════════════════════════════════════
  KISA SA YE? / WHAT IS THIS?
═══════════════════════════════════════════════════════════════════════════

Yon zouti ki pran yon PDF (nan nenpòt lang) epi:
  1. Ekstrè tout tèks la
  2. Tradui li an Kreyòl Ayisyen
  3. Kreye yon liv odyo MP3

A tool that takes a PDF (in any language) and:
  1. Extracts all the text
  2. Translates it to Haitian Creole
  3. Creates an MP3 audiobook

  KÒMANSE RAPID / QUICK START
═══════════════════════════════════════════════════════════════════════════

┌─────────────────────────────────────────────────────────────────────────┐
│  WINDOWS USERS                                                          │
└─────────────────────────────────────────────────────────────────────────┘

  1. Doub-klike sou: run.bat
  2. Script la ap verifye tout bagay epi kòmanse!

  OU BYEN:

  1. Ouvè PowerShell oswa Command Prompt
  2. cd "C:\Users\Fanerlink\OneDrive\Documents\liv odyo\projet_kreyol_IA"
  3. pip install -r requirements.txt
  4. Mete yon PDF nan data\input.pdf
  5. python main.py


┌─────────────────────────────────────────────────────────────────────────┐
│  MAC / LINUX USERS                                                      │
└─────────────────────────────────────────────────────────────────────────┘

  1. chmod +x run.sh
  2. ./run.sh

  OR:

  1. Open Terminal
  2. cd projet_kreyol_IA
  3. pip3 install -r requirements.txt
  4. Place a PDF in data/input.pdf
  5. python3 main.py

  SI W GEN PWOBLÈM / TROUBLESHOOTING

❌ "ModuleNotFoundError"
   → pip install -r requirements.txt

❌ "Pa jwenn PDF / PDF not found"
   → Verifye ke ou gen: data/input.pdf

❌ "out of memory"
   → Fèmen lòt aplikasyon / Close other apps
   → Bezwen omwen 4GB RAM / Need at least 4GB RAM

❌ "Model downloading..."
   → Sa nòmal premye fwa! Tann kèk minit
     This is normal first time! Wait a few minutes

📖 Li INSTALL.md pou plis detay / Read INSTALL.md for more details
