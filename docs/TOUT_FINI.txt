⚙️  KONFIGIRASYON NESESÈ / REQUIRED CONFIGURATION
═══════════════════════════════════════════════════════════════════════════

OBLIGATWA / REQUIRED:
────────────────────────────────────────────────────────────────────────────
1. Kopye config:
   cp cloud_env_template.txt .env

2. Modifye .env:
   GCS_BUCKET_NAME=your-bucket-name

3. Otantifye:
   gcloud auth application-default login


OPSYONÈL / OPTIONAL (pou email):
────────────────────────────────────────────────────────────────────────────
Nan .env:
   SENDER_EMAIL=your-email@gmail.com
   SENDER_PASSWORD=your-app-password
   RECIPIENT_EMAIL=recipient@email.com

🆘 DEPANAJ / TROUBLESHOOTING
═══════════════════════════════════════════════════════════════════════════

Pwoblèm 1: "Bucket not found"
   → Defini GCS_BUCKET_NAME nan .env

Pwoblèm 2: "Authentication failed"
   → gcloud auth application-default login

Pwoblèm 3: "Email pa voye"
   → Itilize Gmail App Password

Pwoblèm 4: "Module not found"
   → pip install -r requirements.txt

Pou pi plis èd:
   python check_setup.py
